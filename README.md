# LyfeLabz

**LyfeLabz is an interactive learning hub for middle school science, built to make lessons feel more like labs than lectures.**  
Designed for students to explore independently and for teachers to use as plug-and-play classroom resources.

## What You'll Find

- Student-friendly layouts optimized for classroom projection or individual devices

## Built With
- HTML and CSS
- GitHub Pages
- Custom domain hosting

## Mission
LyfeLabz exists to create clean, focused, high-engagement science resources that are easy to deploy in classrooms and simple for students to navigate.
